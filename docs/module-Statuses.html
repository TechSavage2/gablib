(static) moveStatus(lo, status, groupId) → {Promise.<({ok: boolean, content: null}|{ok}|*)>}

Description:

  • Convenience function to emulate status moving between home timeline and a group and vice verse, or from one group to another.

    NOTE: This does not perform a real move as mastodon doesn't support moving statuses other than admins 'removing' statuses from groups.

    This function will create a new status based on the original, but with target set to a group or null (if already in a group.)

    If that succeeds the original status is deleted and the new status is returned.
