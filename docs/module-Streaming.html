Methods

(static) getStream(lo, autoReconnectopt) → {Promise.<*>}

Description:

  • Subscribe to the streaming API. It may emit three events:

    stream-event stream-ended stream-error

    You can obtain an event handler by importing gablibEvents from gablib.

    NOTE: This call doesn't return unless a stream-ended event occurred. Therefor we don't recommend using await for this call, but instead the regular Promise.then().catch() approach to allow successive code to execute unhindered.

    A stream-error is also transmitted on error with the error object passed in as the second argument.
